Geomy is a client-side 3D mesh toolbox powered by Three.js. <br> 
- Currently deployed <a href="https://geomy.gker.fr" target="_blank">here</a> 
- Code is accessible <a href="https://github.com/Kiord/geomy" target="_blank">here</a>

This project aims at gathering and deploying the <a href="/projects/mesh_tools" target="_blank">3D mesh tools</a> in a browser with AI assistance (LLMs became incredibly good at writing web apps !)


### Implemented tasks
- 3D inspection
- Landmarking
- Vertex masking
- Vertex segmentation
- Rigid align
